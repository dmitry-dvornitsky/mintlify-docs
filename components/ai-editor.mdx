AI Editor — встроенный AI-ассистент для написания email и LinkedIn сообщений. Открывается кнопкой с цветной звёздочкой ✦ в compose-форме.

## Контекст

AI автоматически использует:
- **Тред** — вся история переписки с контактом
- **Компания и контакт** — данные из Details панели
- **Insights** — информация о продукте клиента (стратегии, описания, конкуренты и т.д.)

## Интерфейс

### Дефолтное состояние

Показывает набор быстрых действий — предустановленные промпты для наиболее частых задач.

Quick actions специфичны для каждого модуля и контекста (Email, LinkedIn, Insights и т.д.). Текущие quick actions временные — нужно переработать под каждый сценарий использования.

Пример текущих quick actions (Email):

| Quick action |
|---|
| Write a professional follow-up email. |
| Make the email more concise and direct. |
| Add a friendly and personalized tone to the email. |
| Write a polite response to decline or postpone. |
| Create a compelling call-to-action for a meeting. |

Внизу — текстовое поле для свободного запроса: `Ask AI to help with your email...`

### После генерации

Показывается сгенерированный текст и кнопки:

| Кнопка | Действие |
|---|---|
| **Insert reply** | Вставить сгенерированный текст в compose-поле |
| **Copy** | Скопировать в буфер |
| **↺ Regenerate** | Сгенерировать ещё раз |

Чат-поле остаётся активным — можно продолжать уточнять и дорабатывать сообщение итеративно.
