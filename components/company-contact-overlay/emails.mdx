Таба Emails — таблица всех email-тредов с компанией или контактом.

Тред принадлежит компании если в нём есть хотя бы один контакт из этой компании.

## Просмотр треда

Клик на тред открывает переписку внутри оверлея. Сообщения в хронологическом порядке, длинные треды сворачиваются с кнопкой "Show N more messages".

## Compose / Reply

Форма отправки внизу страницы треда:

| Поле | Описание |
|---|---|
| From | Email-ящик отправителя — подставляется автоматически |
| To | Получатель. Поддерживает CC/BCC |
| Body | Текстовое поле. "Start typing or select a template" |

**Панель инструментов:**

| Элемент | Описание |
|---|---|
| **✦ AI Editor** | Отдельный компонент для AI-генерации. [Подробнее →](/components/ai-editor) |
| **Quick AI actions** | Быстрые AI-действия без открытия полного редактора |
| **Форматирование** | Bold, Italic, списки, ссылки и др. |
| **Attach** | Прикрепить файл |
| **Create task** | Создать задачу — автоматически подставляется тип `Follow-up in thread` и контекст треда |
| **Send** | Отправить |
| **Schedule** | Запланировать отправку на определённое время |

## Reply / Reply to all

<Warning>
Нужно добавить кнопки **Reply** и **Reply to all** под каждым сообщением в треде (как в Gmail). При клике — поле TO заполняется автоматически из этого письма. Сейчас TO заполняется вручную — риск ошибки и лишний шаг для SDR.
</Warning>
