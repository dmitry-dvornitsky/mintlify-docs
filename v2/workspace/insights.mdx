Insights — база знаний по клиенту: бизнес, продукты, ICP, возражения, материалы. В v2 это **источник контекста для AI**: всё, что генерируют агенты (сиквенсы, ответы, стратегии), опирается на страницы Insights. Чем полнее база — тем лучше работает автоматика.

## Структура

| Раздел | Содержимое |
|---|---|
| **📌 ICP** | Закреплённый стратегический документ — вынесен из общего хранилища, всегда сверху |
| **Client** | Overview, Expectations, Acquisition Channels — страницы о бизнесе клиента |
| **Products** | Вкладка на каждый продукт из Settings: Overview, Pricing, Solution, Competitor Comparison, Objections |
| **Files** | Хранилище с типизацией: Transcripts / Decks / Docs — не свалка, каждая категория отдельно |

<Note>Дублирование v1 устранено: список продуктов живёт только в Settings → Products, страницы знаний о продуктах — только в Insights → Products. AI-стратегии перенесены из Insights в [Strategy Hub](/v2/workspace/strategy-hub).</Note>

## AI-наполнение

Ключевой механизм v2 — знания извлекаются автоматически:

1. AS загружает файл (транскрипт kickoff-звонка, sales-дек, ICP-документ)
2. **Strategist Agent** извлекает факты и раскладывает по страницам: обновляет Overview, добавляет возражения, уточняет Pricing
3. Изменённые страницы получают статус **Draft (AI)** — AS просматривает diff и утверждает
4. Утверждённые страницы становятся доступны AI-генерации

Каждая страница хранит **Source** — из какого файла или звонка пришли данные.

## Статусы страниц

| Статус | Смысл |
|---|---|
| **Draft** | Черновик — не используется AI |
| **Draft (AI)** ✦ | Сгенерировано агентом, ждёт проверки |
| **Approved** | Проверено — используется как контекст AI-генерации |
| **Outdated** | Помечено устаревшим (вручную или при загрузке противоречащего файла) |

<Warning>AI-генерация (сиквенсы, ответы Copilot) использует **только Approved-страницы**. Это защита от галлюцинаций на непроверенных данных.</Warning>

## Редактор

Notion-подобный редактор + AI-чат слева: «перепиши раздел», «добавь возражение», «сравни с конкурентом X». Открывается в панели, не выбивая из контекста модуля. История версий с diff.

## Использование знаний

| Потребитель | Что берёт |
|---|---|
| Strategy Hub | ICP, продукты — черновик стратегии |
| Campaigns | Messaging-контекст для генерации сиквенсов |
| Inbox Copilot | Pricing, Objections — ответы на вопросы лидов |
| Client Portal | Выбранные страницы можно расшарить клиенту на аппрув |
| MCP | `insights.search`, `insights.get_page` — контекст для внешних агентов |

## MCP

`insights.list`, `insights.search` (семантический поиск по базе), `insights.get_page`, `insights.create_page` (статус Draft (AI)), `insights.upload_file`.
