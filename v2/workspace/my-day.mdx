My Day — первый экран после входа. Отвечает на один вопрос: **что требует моего внимания прямо сейчас**. Содержимое собирается автоматически из всех проектов пользователя и адаптируется под роль.

## Блоки

| Блок | Содержимое |
|---|---|
| **Needs your attention** | Единая очередь: непрочитанные ответы лидов, Gates ожидающие approve, просроченные задачи, встречи в ближайший час. Отсортировано по срочности |
| **My KPI** | Личный прогресс за месяц — метрики зависят от роли (см. ниже) |
| **Autopilot summary** | Что агенты сделали автоматически с момента последнего входа — с ссылкой на лог для проверки |
| **My projects** | Проекты пользователя со статусом здоровья (on track / at risk / off track) |
| **Today** | Задачи с дедлайном сегодня из [Tasks](/v2/workspace/tasks) всех проектов |

## KPI по ролям

| Роль | Метрики на дашборде |
|---|---|
| **SDR** | Встречи назначено / план, обработано ответов, среднее время ответа, reply rate своих кампаний |
| **AS** | Здоровье проектов, KPI-выполнение клиентов, ожидающие Gates, дни до дедлайнов |
| **LR** | Контакты в пайплайне, качество (доля Qualified), ожидающие Gates |
| **EDS** | Здоровье ящиков, bounce rate, ящики близкие к лимитам |
| **AD** | Health Score клиентов, ближайшие renewal, открытые эскалации |

## Правила очереди «Needs your attention»

Очередь собирается из событий Event Bus и упорядочивается:

1. Встреча в ближайшие 60 минут
2. Ответ лида с типом Appointment / Interested без реакции > 1 часа
3. Gate, блокирующий пайплайн или кампанию
4. Просроченные задачи
5. Всё остальное по времени поступления

<Note>Клик по любому элементу открывает его в контексте: ответ — в Inbox-треде, Gate — в модалке approve, задача — в Tasks. Из My Day можно закрыть большинство дел, не открывая модули вручную.</Note>

## Для менеджмента

У ролей Operations / C-level вместо My Day точка входа — [Admin → Command Center](/v2/admin/command-center). Переключение доступно в профиле.
