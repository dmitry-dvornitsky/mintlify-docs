Каждый Project соответствует одному клиенту Belkins. Внутри проекта — вся работа по клиенту: стратегии, лиды, кампании, переписка, встречи, знания и настройки.

## Список проектов

**Колонки:** Project, Stage, Health, KPI progress (mini-bar), Active campaigns, Team, Last activity

| Элемент | Описание |
|---|---|
| **Stage** | Этап жизненного цикла: Onboarding / Research / Outreach / Delivery / Renewal / Churned — см. [Client Lifecycle](/v2/overview/client-lifecycle) |
| **Health** | Health Score 0–100 — тот же, что в [Admin → Clients](/v2/admin/clients) |
| **Quick Filters** | My projects / Stage / Health / Account Strategist |

По умолчанию пользователь видит **свои проекты**; полный список доступен по фильтру (для Sales — read-only).

## Дашборд проекта

Один и тот же дашборд видят команда и — в упрощённом виде — клиент в [Client Portal](/v2/workspace/client-portal).

### Блоки

| Блок | Что показывает |
|---|---|
| **KPI** | Встречи за месяц против плана из [Settings → KPI](/v2/workspace/settings/client#kpi) + прогноз темпа (on track / behind / ahead) |
| **Funnel** | Сквозная воронка за период: Contacts → Contacted → Replies → Positive → Meetings. Клик по ступени открывает список лидов на ней |
| **Channels** | Email и LinkedIn рядом: отправлено, reply rate, acceptance rate, встречи по каналу |
| **Active work** | Живые процессы: запущенные пайплайны (с прогрессом), активные кампании, ближайшие встречи |
| **Needs attention** | Гейты, ящики у лимита, старые неотвеченные ответы, просроченные задачи проекта |
| **AI Weekly Summary** | Автоматическая сводка недели: что изменилось, почему, что рекомендуется. Используется и как черновик отчёта клиенту |
| **Project info** | Карточка клиента из Settings → General: команда, timeline, регион, бюджет |

### Табы дашборда

| Таб | Содержимое |
|---|---|
| **Overview** | Блоки выше |
| **Performance** | Гибкий аналитический график: Metric × Group by × Split by × Date range |
| **Campaigns** | Таблица кампаний проекта с агрегатами (Contacted, Reply Rate, Reply Types donut, Meetings) |
| **Report** | Архив еженедельных AI-отчётов, отправленных клиенту, с возможностью редактирования перед отправкой |
