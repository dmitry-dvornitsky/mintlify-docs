Command Center — точка входа C-level и Operations. Четыре вопроса на одном экране: **сколько зарабатываем, доставляем ли обещанное, какая маржа, где риски**.

## Верхняя панель (North Star)

| Карточка | Расчёт |
|---|---|
| **MRR / ARR** | Сумма активных Recurring-подписок всех клиентов + тренд MoM |
| **Meetings vs Plan** | Сумма Completed-встреч / сумма KPI Goals всех проектов за месяц |
| **Gross Margin** | (Выручка − внешние расходы: инструменты, LLM) / выручка |
| **Active Clients** | Количество + сколько в статусе at risk (Health < порога) |

## Блоки

| Блок | Что показывает |
|---|---|
| **Delivery Pace** | Встречи против плана по неделям + прогноз закрытия месяца. Красный прогноз — главный ранний сигнал недопоставки |
| **Revenue by Client** | Клиенты по выручке с Health Score рядом — видно, какая выручка под риском |
| **Alerts** | Активные пороговые алерты: health-падения, overdue-инвойсы, capacity команды, деградация deliverability |
| **AI Weekly Brief** | Понедельничный AI-дайджест: что изменилось, почему, какие решения назрели. Отправляется также в email/Slack |
| **Funnel (org-wide)** | Сквозная воронка всех клиентов за период: contacted → replies → positive → meetings — здоровье модели доставки в целом |

## Фильтры

Период (месяц по умолчанию), сегмент клиентов (индустрия, размер контракта, AD), сравнение с прошлым периодом.

## Дрилл-даун

Каждая цифра кликабельна: MRR → [Finance](/v2/admin/finance), клиент → карточка в [Clients](/v2/admin/clients), delivery pace → список проектов, отстающих от плана, capacity-алерт → [Team](/v2/admin/team-performance).

## MCP

`admin.metrics` (набор North Star метрик за период), `admin.alerts`, `admin.weekly_brief` — C-level может спрашивать состояние компании у ассистента в чате, данные придут отсюда.
