Admin — поверхность для менеджмента и C-level. Если Workspace отвечает на вопрос «что делать по этому клиенту», Admin отвечает на «как дела у компании»: по всем клиентам, всей команде, всем деньгам и всей автоматике сразу.

Доступ определяется управленческой ролью — см. [Роли и доступы](/v2/overview/roles#управленческие-роли-admin-модуль).

## Принцип: ноль ручной отчётности

Каждая метрика Admin считается из живых данных платформы — тех же, с которыми работает команда. Нет выгрузок, нет «собери цифры к понедельнику»: если встреча проставлена в Meetings, она уже в Command Center; если инвойс просрочен — он уже в алертах.

**Автоматизация Admin-уровня** (см. [Automation → Workflows](/v2/automation/workflows)):
- Понедельничный AI-дайджест C-level: неделя в цифрах, аномалии, решения к принятию
- Алерты порогов: Health Score упал, KPI-темп ниже плана, маржа клиента ниже X%, ящики деградируют массово
- Месячный financial close: сверка инвойсов Wave ↔ платформа
