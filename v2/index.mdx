Belkins Home 2.0 — вторая версия внутренней платформы Belkins. Если v1 заменила внешние инструменты (Reply.io, Expandi, HubSpot, Planhat) единым интерфейсом, то v2 меняет саму модель работы: **платформа выполняет работу, команда управляет качеством**.

## Три принципа v2

<CardGroup cols={3}>
  <Card title="AI-native" icon="sparkles">
    Каждый процесс — поиск лидов, написание писем, ответы, отчёты — выполняется AI-агентами. Человек включается только в точках контроля (Gates).
  </Card>
  <Card title="Everything is a Tool" icon="plug">
    Любое действие в UI доступно как MCP-инструмент. Один и тот же API используют интерфейс, автоматизации и внешние AI-агенты.
  </Card>
  <Card title="Full visibility" icon="chart-line">
    Admin-модуль даёт менеджменту и C-level полную картину: клиенты, деньги, команда, качество, работа агентов — без выгрузок и ручных отчётов.
  </Card>
</CardGroup>

## Что изменилось относительно v1

| Область | v1 | v2 |
|---|---|---|
| Поиск лидов | 6 ручных операций с ожиданием между ними | Research Pipeline — один запуск, автоматическое выполнение |
| Написание писем | SDR пишет / AI помогает | AI пишет, SDR утверждает; Autopilot для типовых ответов |
| Ответы лидам | SDR отвечает вручную на всё | AI Copilot готовит черновики; типовые ответы — автоматически |
| Requests | Журнал ручных операций | Заменён на Pipeline History внутри Strategy |
| Отчётность менеджменту | Выгрузки и вопросы в Slack | Admin → Command Center в реальном времени |
| Финансы | Billing внутри Settings проекта | + Admin → Finance по всем клиентам, каталог сервисов, маржа |
| Клиент | Client View — ограниченный дашборд | Client Portal — дашборд, встречи, аппрувы, чат с командой |
| API | Нет | MCP-сервер + REST API поверх единого слоя действий |
| Аудит | Нет | Полный Audit Log: люди и агенты, каждое действие |
