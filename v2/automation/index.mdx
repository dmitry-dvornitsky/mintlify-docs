## Принципы безопасной автономии

1. **Один слой действий.** Агент не может сделать ничего, что недоступно человеку с той же ролью — UI, агенты и MCP ходят через общий Action Layer.
2. **Автономия — настройка, не свойство.** Каждый агент работает на уровне L1–L4 (см. [Роли](/v2/overview/roles#ai-агенты)), уровень меняется per-project и автоматически понижается при деградации качества.
3. **Gate по умолчанию для необратимого.** Отправка вовне (письмо незнакомого типа, инвойс, запуск кампании) требует approve, пока статистика не заслужила автономию.
4. **Всё видно и всё отменяемо.** Audit Log + reasoning trace + Undo для действий Autopilot + kill switch в [Admin → System](/v2/admin/system#automation-monitor).
5. **Бюджеты.** LLM-расходы агентов лимитированы per-project и per-agent; превышение — стоп и алерт.

## Как это масштабирует агентство

| Работа | Раньше | С Automation |
|---|---|---|
| Найти 1000 контактов | Дни LR-времени | Часы, LR — только Gates |
| Ответить на 100 реплаев | День SDR | 60 закрывает Autopilot, 40 — черновики Copilot |
| Собрать weekly-отчёты по 30 клиентам | Часы AS | Автоматически, AS проверяет |
| Выставить инвойсы месяца | Часы Finance | Автоматически, Finance аппрувит пачкой |
| Онбординг знаний нового клиента | Неделя AS | AI извлекает из файлов, AS утверждает |
