Таблица — главный UI-компонент приложения. Используется практически везде: Projects, Leads, Campaigns, Reporting и других модулях.

## Структура

### Header

Слева:
- **Sort** — сортировка по колонкам, в том числе multi-sort (по нескольким колонкам)
- **Filters** — расширенная фильтрация. Когда фильтры применены показывает `Filtered by {Field}`, при нескольких — добавляет `+N`
- **Quick Filters** — быстрые предустановленные фильтры

Справа:
- **View** — управление отображением: видимость колонок, порядок колонок
- **Action** — действия над данными, набор кнопок зависит от таблицы (часто — Create)

### Колонки

По клику на заголовок колонки открывается меню:

| Действие | Описание |
|---|---|
| Sort Ascending | Сортировка по возрастанию |
| Sort Descending | Сортировка по убыванию |
| Move Left | Переместить колонку влево |
| Move Right | Переместить колонку вправо |
| Hide from view | Скрыть колонку |

Также поддерживается изменение ширины колонки перетаскиванием границы заголовка.

### Body

Первая колонка — чекбоксы для выбора строк. При клике на главный чекбокс в хедере появляется меню:

| Опция | Статус |
|---|---|
| Select page | ✅ Есть |
| Select all table | ✅ Есть |
| Select range | 🔲 Нужно добавить |

При выборе строк внизу экрана появляется **Bulk Action Bar** — панель с действиями для работы с выбранными записями. Набор действий зависит от таблицы.

В некоторых таблицах при наведении на строку появляются **Quick Actions** прямо в строке.

### Footer

Слева:
- Общее количество записей в таблице

Справа:
- **Rows per page** — селектор количества строк на странице
- **Pagination** — навигация по страницам
