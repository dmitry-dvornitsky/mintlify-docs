Tasks — список задач по проекту. Задачи создаются вручную или автоматически системой.

## Таблица

Колонки: **Title**, **Status**, **Task Type**, **Assigned To**.

Таблица разделена на три секции:

| Секция | Описание |
|---|---|
| Overdue | Задачи с просроченным дедлайном |
| Scheduled | Задачи с будущим дедлайном |
| Done | Выполненные задачи |

## Создание задачи

Кнопка **Add task** открывает поп-ап с полями:

| Поле | Описание |
|---|---|
| Task type | Тип задачи (см. ниже) |
| Assign to | Исполнитель. По умолчанию — текущий пользователь |
| Title | Название задачи |
| Due date | Дата дедлайна |
| Due time | Время дедлайна |
| Timezone | Часовой пояс дедлайна — подставляется автоматически из настроек пользователя |
| Description | Описание задачи |

## Типы задач

Тип определяет поведение задачи — ручная или автоматическая.

### Task *(default)*
Обычная задача, создаётся и закрывается вручную.

### Follow-up in thread
Автоматическая задача. Отмечается как **Done** автоматически когда SDR отправляет письмо в существующий тред с контактом.

### Follow-up in new thread
Автоматическая задача. Отмечается как **Done** автоматически когда SDR отправляет письмо в **новый** тред — начинает свежую переписку с контактом.

<Info>
  Типов задач будет больше — планируются типы связанные с поиском лидов и другими процессами.
</Info>
