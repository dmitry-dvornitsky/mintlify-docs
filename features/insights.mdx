Insights — база знаний по проекту. Хранит всю информацию о клиенте, его бизнесе и продуктах. Каждая страница выглядит как Notion-документ.

## Структура сайдбара

### 1. Storage

Файловое хранилище материалов от клиента:
- Транскрипты звонков
- Sales-деки
- ICP (Ideal Customer Profile)
- Другие файлы клиента

### 2. Client (Business)

Структурированная информация о бизнесе клиента. Разделы внутри:

| Раздел | Содержание |
|---|---|
| Overview | Общая информация о компании |
| Expectations | Ожидания клиента от сотрудничества |
| Acquisition Channels | Каналы привлечения клиентов |
| Products | Продукты клиента |

### 3. Product (× N)

Отдельная вкладка для каждого продукта клиента — создаётся автоматически из [Settings → Products](/features/settings#products). Разделы внутри каждого продукта:

| Раздел | Содержание |
|---|---|
| Overview | Описание продукта |
| Pricing | Ценообразование |
| Solution | Что решает продукт |
| Competitor Comparison | Сравнение с конкурентами |
| Objections | Типичные возражения и ответы на них |
| **Strategies** | **Требует отдельного описания — см. TODO** |

## Создание страниц

Внутри каждого раздела — таблица страниц и кнопка **Add page**.

При нажатии **Add page** открывается редактор:
- **Слева** — чат с AI: можно попросить сгенерировать или улучшить контент
- **Справа** — текстовый редактор в стиле Notion

Можно писать вручную, через AI или комбинировать.
