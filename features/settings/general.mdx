## Информация о компании

| Поле | Описание | Статус |
|---|---|---|
| Name | Название клиента | ✅ |
| Slug | URL-friendly идентификатор проекта, автогенерируется из Name | ✅ |
| Size | Размер компании клиента | ✅ |
| Industry | Индустрия | ✅ |
| Country | Страна | ✅ |
| State | Штат / регион | ✅ |
| Timezone | Часовой пояс клиента — важно для таймингов outreach | 🔲 Нужно добавить |
| Storage Link | Ссылка на Google Drive по проекту | ✅ |
| Website | Сайт клиента | ✅ |
| Contact Person | Главный стейкхолдер на стороне клиента | 🔲 Нужно добавить |

## Контракт

| Поле | Описание |
|---|---|
| Customer Since / To | Период работы с клиентом |
| Contract Pilot | Длительность пилотного периода в месяцах (например, 3). В это время команда нарабатывает рабочие стратегии для проекта. |
| Contract Recurring | Рекаррин-период в месяцах (1, 3, 6 или любое другое число) |
| Contract Specifics | Детали контракта в свободной форме |
