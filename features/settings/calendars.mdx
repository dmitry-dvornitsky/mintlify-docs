Минимум один подключённый календарь необходим для работы модуля [Meetings](/features/meetings).

## Таблица календарей

Список всех подключённых календарей. Можно подключить любое количество.

## Добавление календаря

Кнопка **Add calendar** открывает поп-ап с выбором типа:

| Тип | Способ подключения |
|---|---|
| Google Calendar | OAuth — редирект на авторизацию Google |
| Microsoft Calendar | OAuth — редирект на авторизацию Microsoft |
| Calendly | API key — вставляется вручную |
