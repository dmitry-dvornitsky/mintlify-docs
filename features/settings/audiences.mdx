Audiences — именованные наборы должностей (job titles), которые команда определяет один раз и затем переиспользует при поиске контактов.

Вместо того чтобы вручную вводить должности при каждом [Find Contacts](/features/leads#find-contacts-in-companies), выбираешь готовую аудиторию из списка.

Таблица с кнопкой **Add Audience**.

## Add Audience

| Поле | Описание |
|---|---|
| Audience Name * | Название аудитории, e.g. "C-Level Executives" |
| Description * | Описание — кто эти люди и чем уникальны. Используется AI для квалификации контактов: AI определяет какие должности подходят под описание, а какие нет |
| Job Titles * | Должности тегами. Вводить и нажать Enter |
| Exclude Job Titles | Должности для исключения. По умолчанию уже заполнены: Assistant, Associate, Junior, Intern, Student, Trainee, Office |

Кнопка **✦ Improve with AI** в поле Description — AI улучшает формулировку описания.

## Использование

Audiences выбираются в двух полях при поиске контактов:
- **First Priority** — основная целевая аудитория
- **Secondary Priority** — запасная (если контакты первой аудитории не найдены)

Связано с: [Leads → Find Contacts in Companies](/features/leads#find-contacts-in-companies)

<Warning>Флоу работы с Audiences планируется переработать в рамках общего редизайна стратегии поиска лидов.</Warning>
