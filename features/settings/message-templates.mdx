Шаблоны сообщений которые SDR вставляет в переписку в один клик — без необходимости печатать с нуля. Работают в обоих каналах: **Email** и **LinkedIn**.

Таблица с кнопкой **Add Template**.

**Колонки:** Name, Updated By

## Add New Template

| Поле | Описание |
|---|---|
| Template Name | Название шаблона для поиска в списке |
| Template Content | Текст шаблона. Поддерживает форматирование (кнопка **A**) и AI-генерацию (кнопка **✦**) |

## Использование

Шаблоны доступны в редакторе сообщений в [Company/Contact Overlay](/components/company-contact-overlay) — в табах Email и LinkedIn.
