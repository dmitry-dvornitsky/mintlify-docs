Раздел содержит финансовую информацию по клиенту, организован в 3 таба.

## Billing Information

Контактные, юридические данные и настройки биллинга для выставления инвойсов:

| Поле | Описание |
|---|---|
| Legal name | Юридическое название компании |
| Location | Адрес |
| Email | Email для инвойсов |
| First name / Last name | Контактное лицо |
| Wave customer | Привязка проекта к клиенту в Wave (интеграция для выставления инвойсов) |
| Customer NET (days) | Срок оплаты инвойса в днях (например, 14 = NET 14) |

## Services

Список сервисов, подключённых к проекту. Поддерживает сортировку и фильтрацию, набор колонок настраивается через **View**.

| Колонка | Описание |
|---|---|
| Service | Название сервиса |
| Type | Тип подписки: Recurring (фиолетовый бейдж) или One-time |
| Status | Статус сервиса: Active (зелёный) |
| Service Price | Стоимость за цикл |
| Billing Cycle (months) | Частота выставления инвойса (например, Every month) |
| Period | Даты действия сервиса (start – end date) |

Новый сервис добавляется через кнопку **Add service**.

Справочник доступных сервисов (название, цена по умолчанию) ведётся в разделе **Admin** — менеджеры управляют каталогом там. При выборе сервиса из дропдауна его цена подставляется автоматически, но может быть изменена вручную.

### Типы подписок

#### Recurring

Повторяющаяся подписка — инвойс выставляется автоматически каждый billing cycle.

| Поле | Описание |
|---|---|
| Service | Сервис из каталога (цена подставляется автоматически, можно изменить) |
| Price per month | Стоимость в месяц |
| Currency | Валюта (по умолчанию USD) |
| Billing cycle (months) | Как часто выставляется инвойс (например, 1 месяц = ежемесячно) |
| Contract length (months) | Общая длительность контракта (например, 12 месяцев) |
| Start date | Дата первого инвойса |
| End date | Рассчитывается автоматически: start date + contract length |
| Notes | Примечания к инвойсу |

**Setup fee** — опциональный разовый сбор. Включается чекбоксом, сумма вводится вручную. Добавляется к первому инвойсу.

**Discount** — опциональная скидка, применяется по Amount или Percent. При добавлении скидки указывается **Discount duration**:

| Вариант | Описание |
|---|---|
| One-time (first billing only) | Скидка только на первый инвойс |
| Limited duration (months) | Скидка на указанное количество месяцев |
| Till end of subscription | Скидка действует на весь срок контракта |

Справа в модале отображается **Preview** — список всех инвойсов с датами и суммами, включая setup fee и дискаунт, в реальном времени.

#### One-time

Разовый платёж — генерируется один инвойс.

| Поле | Описание |
|---|---|
| Service | Сервис из каталога (цена подставляется автоматически, можно изменить) |
| Price | Стоимость |
| Currency | Валюта (по умолчанию USD) |
| Invoice date | Дата выставления инвойса |
| Notes | Примечания к инвойсу |

Setup fee и Discount доступны аналогично Recurring.

## Invoices

Таблица инвойсов по проекту. Поддерживает сортировку, фильтрацию и настройку колонок через **View**.

| Колонка | Описание |
|---|---|
| Invoice Title | Название сервиса, к которому относится инвойс |
| Status | Статус инвойса (см. ниже) |
| Total | Итоговая сумма инвойса |
| Paid | Оплаченная сумма (— если не оплачен) |
| Issue Date | Дата выставления инвойса |
| Due | Дата оплаты (Issue Date + Customer NET) |
| Source | Откуда пришёл инвойс: **Subscription** (создан в системе) или **Synced** (синхронизирован из Wave) |

### Статусы инвойсов

| Статус | Описание |
|---|---|
| Draft | Черновик, ещё не отправлен |
| Created | Создан и готов к отправке |
| Paid | Оплачен |
| Overdue | Просрочен (дата Due прошла, оплата не поступила) |

### Отправка в Wave

При выборе одного или нескольких инвойсов через чекбокс в тулбаре появляется кнопка **Send to Wave**. Wave — внешний сервис для выставления инвойсов клиентам, подключён через API. После отправки инвойс синхронизируется между системой и Wave.
