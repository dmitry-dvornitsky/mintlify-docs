Раздел содержит финансовую информацию по клиенту, организован в 4 таба.

## Billing Information

Контактные и юридические данные для выставления инвойсов:

| Поле | Описание |
|---|---|
| Legal name | Юридическое название компании |
| Location | Адрес |
| Email | Email для инвойсов |
| First name / Last name | Контактное лицо |
| NET | Срок оплаты инвойса в днях (например, NET 30 = 30 дней) |

## Services

Подписки и one-time сервисы, предоставляемые клиенту.

## Invoices

Таблица инвойсов для клиента.

## Churn & Downgrade

История чарнов и даунгрейдов клиента. Записи добавляются вручную для трекинга сумм и дат.
