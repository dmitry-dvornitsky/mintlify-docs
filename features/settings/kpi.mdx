## Добавление KPI

Кнопка **Add KPI** открывает поп-ап с полями:
- **Month** — месяц
- **Year** — год
- **Goal** — целевое количество встреч

## Таблица

| Колонка | Описание |
|---|---|
| Month | Месяц |
| Year | Год |
| Goal | Плановое количество встреч |
| Result | Количество встреч со статусом **Completed** — подтягивается автоматически из модуля [Meetings](/features/meetings) |
| % | Процент выполнения KPI (Result / Goal) — вычисляемая колонка |
| Updated by | Кто последний редактировал Goal |

Удаление записей недоступно намеренно — для сохранения истории.

## Действия по строке

| Действие | Описание | Доступ |
|---|---|---|
| Edit | Редактировать Goal | Account Director, Account Strategist, Менеджеры с Admin доступом |
| KPI History | Поп-ап с историей изменений — кто и что менял | Все |
| Meetings History | История встреч за месяц, к которому относится KPI | Все |
