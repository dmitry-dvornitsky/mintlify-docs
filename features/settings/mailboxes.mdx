Mailboxes — таблица подключённых email-ящиков команды. Используются для отправки и получения писем в Inbox и Overlay.

Таблица с кнопкой **Add Mailbox**.

## Колонки таблицы

| Колонка | Описание |
|---|---|
| Email | Адрес ящика |
| Sender Name | Имя отправителя которое видит получатель |
| Provider | Gmail / Outlook (с иконкой провайдера) |
| Status | **Active** / другие статусы |
| Usage | Дневной лимит: отправлено / лимит (например 50/70). Прогресс-бар: жёлтый — умеренная загрузка, красный — близко к лимиту |
| Added | Дата подключения ящика |

## Подключение

| Способ | Статус |
|---|---|
| Google (Gmail) | OAuth |
| Microsoft (Outlook) | OAuth |
| SMTP | Планируется |
