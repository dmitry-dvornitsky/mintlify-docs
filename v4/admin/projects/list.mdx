Портфель проектов в Admin. По умолчанию показывает **мои проекты** (где пользователь ответственный AS/AD); весь портфель — по фильтру. Delivery-команда этот экран не видит — её проекты живут в [Home](/v4/workspace/home).

## Кто использует

AS, AD, Ops, C-level. Sales — read-only по своим клиентам. Заглушённые для роли действия не показываются вовсе (не disabled-кнопки).

## Как работает

Таблица проектов, клик по строке → [дашборд проекта](/v4/workspace/dashboard) в Workspace.

| Project | Stage | Health | KPI | Campaigns | Team | Last activity |
|---|---|---|---|---|---|---|
| Acme Corp | Active | 82 | 6 / 12 | 3 | KM · AC · +2 | 2 часа назад |
| BuildCo | Onboarding | — | 0 / 8 | 0 | KM | вчера |
| FinPeak · ⚠ инвойс | Paused | 45 | 2 / 10 | 1 | ML | 14 дней назад |

Stage — по [жизненному циклу](/v4/admin/projects/lifecycle); Health — скор 0–100, тот же что в [Admin → Clients](/v4/admin/clients); KPI — встречи месяца против плана.

**Quick Filters:** My projects (по умолчанию) · Stage · Health · Account Strategist.

## Правила и эдж-кейсы

| Ситуация | Поведение |
|---|---|
| Брошенный визард создания | Проект виден в списке с бейджем **Draft**, клик продолжает визард с того же шага |
| Архивные проекты | Скрыты по умолчанию; фильтр Stage → Archived показывает их (read-only) |
| Просроченный инвойс | Красный бейдж на строке проекта; виден AS/AD и выше |
| У AS/AD ноль своих проектов | Empty state с CTA «Create project»; фильтр «все проекты» остаётся доступен |
| Ops/C-level без своих проектов | Дефолтный фильтр — «все проекты», а не «мои». Это их вход в любой проект: клик открывает дашборд в read-only с баннером «Вы не в команде проекта» ([Layout](/v4/layout)) |
| Проект без активности 14+ дней | Строка помечается предупреждением — стоит проверить |
| 100+ проектов | Пагинация + поиск по названию всегда виден |

## Чеклист функционала

- [ ] Таблица: Project, Stage, Health, KPI, Campaigns, Team, Last activity
- [ ] Дефолтный фильтр «My projects», переключение на все
- [ ] Quick Filters: Stage, Health, Account Strategist
- [ ] Сортировка по любой колонке (дефолт — Last activity)
- [ ] Бейджи состояний: Draft, Paused, просроченный инвойс, нет активности
- [ ] Архивные скрыты по умолчанию, доступны фильтром
