Кампейн — исполнительный слой [стратегии](/v4/workspace/strategies): своего месседжинга у него нет. Формула: **снапшот угла стратегии + порция лидов + сиквенс + ящики и темп**. Кампейн без стратегии невозможен — AI-генератору не из чего писать.

Два типа ([D-029](/v4/decisions)): **AI Generated** — в шагах сиквенса промпты, AI пишет уникальное письмо каждому лиду; **Static** — в шагах статичные темплейты с переменными и спинтаксом, для клиентов с жёстким контролем месседжа. Тип строгий: миксовать шаги нельзя.

Цепочка контента насквозь: Документы → Стратегия («Что говорим») → Кампейн (сиквенс) → письмо лиду. У AI-черновиков в [Inbox](/v4/workspace/inbox) виден источник: «CFO UK · What we say v3».

## Кто использует

| Роль | Что делает |
|---|---|
| **AS** | Создаёт, аппрувит запуск (Gate), паузит |
| **CW** | Пишет и правит тексты: промпты шагов / темплейты |
| **SDR** | Работает с ответами в Inbox; следит за своими кампейнами |
| **EDS** | Здоровье ящиков и лимиты ([Sending](/v4/workspace/campaigns/sending)) |

Name включает стратегию-родителя. Метрики живые — трекаем только ответы ([D-032](/v4/decisions)), открытия не трекаем. Sending — мини-индикатор ёмкости (ящики × health). Статусы: Draft / On review / Active / Paused / Completed / Archived.

### Детальная — четыре таба

**Overview** (метрики + разбивка по шагам: какое касание приносит ответы) · **Sequence** (шаги, правка) · **Leads** (набор, статусы по каждому) · **Sending** (пул ящиков, темп).

### Жизненный цикл

```
Draft → On review (Gate: сэмплы/рендер + чеклист) → Active ⇄ Paused → Completed → Archived
```

| Действие в Active | Поведение |
|---|---|
| Досыпать лидов | Батч с теми же проверками: эксклюзивность (D-026), ревалидация 90 дней (D-027), DNC |
| Править сиквенс | Только неотправленное: контакт на шаге 2 из 5 получит новые версии шагов 3–5; отправленное не трогается |
| Pause | Немедленно; запланированные письма замирают, треды в Inbox живут |
| Completed | Все касания исчерпаны; контакты освобождаются (эксклюзивность снимается) |
| Archive | После Pause/Completed; статистика **остаётся** в агрегатах проекта и отчётах ([D-007](/v4/decisions)) |

## Правила и эдж-кейсы

| Ситуация | Поведение |
|---|---|
| Лид ответил | Секвенция для него останавливается немедленно (любой шаг, любой канал), тред уходит в [Inbox](/v4/workspace/inbox) |
| Лид попал в DNC на лету | Немедленное исключение, запланированные письма отменяются (правило [Do Not Contact](/v4/workspace/settings/dnc)) |
| Все ящики кампейна встали | Авто-пауза + алерт AS и EDS + баннер в проекте (см. [Mailboxes](/v4/workspace/settings/mailboxes)) |
| Лиды кончились (все прошли сиквенс) | Кампейн → Completed; карточка в Home: «продолжить? Find more в стратегии» |
| Встреча забукана | Секвенция контакта останавливается; meetings automation — пауза остальных контактов компании (THINK) |
| Правка стратегии-родителя | Кампейн не меняется — он на снапшоте (D-028); новая версия угла — для новых кампейнов |
| Bounce rate шага выше порога | Алерт EDS + автопауза отправки с этого ящика; письма перераспределяются |
