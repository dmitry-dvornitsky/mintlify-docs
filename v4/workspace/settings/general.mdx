Кто клиент и на каких условиях работаем. Заполняется при создании проекта, здесь правится.

## Кто использует

Правка — уровни **1–2** ([D-051](/v4/decisions)). Остальные читают.

## Как работает

**Компания:** Name · Slug (автогенерируется из Name, можно менять) · Size · Industry · Country · State · Storage Link (ссылка на Google Drive проекта) · Website.

**Контракт:** Customer Since / To · Pilot (мес.) · Recurring (мес., 1/3/6/…) · Specifics (свободный текст).

**Состояние проекта** — два селекта (списки будут заменены — [D-054](/v4/decisions)):
- **Project Status** (обязательный): Pending · In Progress · At Risk · Done · Paused · Churn
- **Phase**: Active Ongoing · Onboarding · Adoption · Active Loyal · Active Struggling · Predicted Working Off · Working Off · Churn · Predicted Churn · Delayed Start · Delayed Payment – Outreach on hold · On Hold

## Правила

- Почти все поля обязательные.
- Slug можно менять после создания проекта — с предупреждением, что URL проекта изменятся.
- State заполняется для любой страны, не только US.
- Storage Link — просто внешняя ссылка, интеграции с Drive нет.
- Конец пилота ничего не запускает — Pilot чисто информационное поле.

| Поле | Описание |
|---|---|
| Since | Начало периода |
| To | Конец; — если период текущий (открыт) |
| Type | Pilot / Recurring |
| Cycle | Если Recurring: 1 / 3 / 6 / 12 мес. |
| Specifics | Свободный текст, правится на месте (не версионируется) |

**Одна форма «Новый период», два триггера:**
- **Status → Closed** — текущий период закрывается **автоматически** (To = дата смены статуса), без формы: закрывать нечего.
- **Status Closed → Active** (клиент вернулся) — система **сама открывает форму** нового периода: Since по умолчанию = сегодня, обязательно указать Type/Cycle/Specifics — условия возврата не угадать.
- **Смена условий без ухода** (Pilot стал Recurring, поменялась Cycle, Status остаётся Active) — **вручную**, кнопка **+ New period** доступна в любой момент; та же форма закрывает текущую строку (To = сегодня) и открывает новую.

Specifics правится в текущей строке без создания новой (это заметки, не коммерческое условие). Customer Since (сводкой над таблицей) = Since первой строки. Правка — уровни 1–2.

Превью периодов:

| Since | To | Type | Cycle | Specifics |
|---|---|---|---|---|
| 03.2024 | 09.2024 | Pilot | — | — |
| 09.2024 | 06.2025 | Recurring | 3 мес. | Discount 10% до конца квартала |
| 01.2026 | — (текущий) | Recurring | 1 мес. | — |
