Кастомные поля проекта для компаний и контактов. Заполняются AI по инструкции — используются и как данные, и как критерии фильтрации ([D-074](/v4/decisions) — сюда влились критерии квалификации).

## Кто использует

Создание колонок и запуск Enrich — уровни **1–3**.

## Как работает

Два таба — **Companies** и **Contacts**. Таблица с кнопкой **Add column**. Форма:

| Поле | Описание |
|---|---|
| Name * | Название, например «Tech Stack», «Produces tin cans» |
| Type * | Text · Long text · Number · Date · True/False · Single choice · Multiple choice |
| Description | Что хранится в поле |
| AI enrichment instructions | Что и откуда искать: «главные языки разработки по сайту и вакансиям» |

После создания колонка появляется в таблицах Leads → Companies / Contacts как поле.

**Enrich** — запуск AI-обогащения: выбрать компании/контакты чекбоксами → тулбар **Enrich custom columns** → AI ищет по сайту компании / LinkedIn контакта и заполняет поля. История запусков — в Requests (переезжает в историю).

## Правила

- Тип колонки фиксируется при создании.
- AI берёт данные с сайта компании или из LinkedIn-профиля контакта.
- Квалификация выражается колонками: булев критерий = True/False колонка, порог = Number; вердикт «подходит» = фильтр по колонкам ([D-074](/v4/decisions)).
