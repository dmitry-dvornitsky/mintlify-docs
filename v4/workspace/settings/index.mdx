Настройки проекта — нижний пункт проектного [сайдбара](/v4/layout). Принцип V4: **частое настраивается по месту** (inline из форм стратегий, кампаний, оверлея), Settings — библиотека переиспользуемых объектов и редкие настройки.

Layout — [паттерн рельсы D-010](/v4/decisions): глобальный сайдбар сжат в иконки, панель модуля — навигация по пяти разделам (видимость по ролям) с поиском «Find a setting…» и алерт-бейджами; Details-панели нет — контент занимает всю ширину. Скрин — на [странице Каналов](/v4/workspace/settings/channels).

## Кто использует

| Раздел | Кто работает |
|---|---|
| [Клиент и KPI](/v4/workspace/settings/client) | AS, AD |
| [Команда](/v4/workspace/settings/team) | AS, AD |
| [Каналы](/v4/workspace/settings/channels) | EDS (здоровье), AS (подключение) |
| [Таргетинг](/v4/workspace/settings/targeting) | AS, LR; SDR — чтение |
| [Биллинг](/v4/workspace/settings/billing) | AD, Finance, C-level — остальные раздел не видят |

Недоступные роли разделы не видят вовсе (не disabled). Полная матрица — [Роли и права](/v4/platform/roles).
