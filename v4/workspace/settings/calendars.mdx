Календари, через которые букаются встречи и предлагаются слоты ([Meetings](/v4/workspace/meetings)).

## Кто использует

Подключение и настройка — уровни **1–3** ([D-068](/v4/decisions)).

## Как работает

Таблица подключённых календарей, количество не ограничено. Кнопка **Add calendar** — поп-ап с выбором типа:

| Тип | Способ подключения |
|---|---|
| Google Calendar | OAuth — редирект на авторизацию Google |
| Microsoft Calendar | OAuth — редирект на авторизацию Microsoft |
| Calendly | API key — вставляется вручную |

## Правила

- Минимум один подключённый календарь необходим для работы [Meetings](/v4/workspace/meetings).
