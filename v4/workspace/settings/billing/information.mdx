Таб экрана [Billing](/v4/workspace/settings/billing). Реквизиты клиента для выставления инвойсов.

## Как работает

Поля:

| Поле | Описание |
|---|---|
| Legal name | Юридическое название компании |
| Location | Адрес для инвойсов |
| Email | Email, на который уходят инвойсы |
| First / Last name | Контактное лицо по оплате |
| Wave customer | Связка проекта с клиентом в Wave (для синка инвойсов) |
| Customer NET | Дней на оплату (например, 14 = NET 14); Due Date инвойса = Issue + NET |

## Правила

- **Реквизиты хранятся на проекте** (Billing Information — источник правды), клиент = один проект ([D-052](/v4/decisions)); **Wave customer** лишь связывает проект с клиентом в Wave, не отдельное хранилище.
- Юридические поля (Legal name, billing address, invoice email) намеренно **отдельны** от операционного профиля в [General](/v4/workspace/settings/general) — они могут отличаться.
