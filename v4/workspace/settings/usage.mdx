Во что обходится доставка по проекту — расход на внешние инструменты и AI (Generect, OpenAI, Apollo и др.). Это **внутренние деньги**, парные к клиентской выручке в [Billing](/v4/workspace/settings/billing): выручка − Usage = маржа ([D-083](/v4/decisions)). Сквозная картина по всем проектам — [Admin · Finance](/v4/admin/finance).

## Кто использует

Уровень **1 (Admin)** и **AS (2)** — как Billing. Остальные роли раздел не видят.

## Как работает

**Фильтры сверху:** Module (All / Insights / Campaigns / Leads / Inbox / Tasks / Meetings / Other) · User (All / System / конкретные) · Date range · **Export**.

**Total Spent** — суммарный расход за выбранный период.

**Daily Usage** — bar-chart по дням с переключателем группировки **Tool / Feature**:
- **Tool** — цвет = инструмент (Apollo, Generect, Gsuite, Hunter, OpenAI, Quickemail, Rapidapi);
- **Feature** — цвет = фича внутри инструмента.

**Breakdown** — таблица итогов за период (переключатель Tool / Feature): Tool/Feature · Share (полоса) · Credits · Cost · %.

**Request Log** — лог каждого API-вызова: Tool · Feature · Credits · Cost / Credit · Total Cost · Date.

Инструменты и за что списывается:

| Инструмент | Назначение |
|---|---|
| Generect | Парсинг LinkedIn-профилей и данных компаний |
| OpenAI | AI: AI Editor, Company Analysis, Column Enrichment, Campaigns |
| Hunter | Поиск и верификация email |
| Apollo | LinkedIn по email, Phone Enrichment |
| Quickemail | Email delivery / warming |
| Rapidapi | Дополнительные API-запросы |
| Gsuite | Google Calendar / Gmail |

## Правила

- Расход **без привязки к проекту** сюда не попадает — он в [Admin · Finance](/v4/admin/finance) (общие расходы).
- Cost — в деньгах ($); Credits — сырое потребление инструмента.
