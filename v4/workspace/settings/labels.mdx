Метки для классификации тредов в [Inbox](/v4/workspace/inbox). Двухуровневая иерархия: **Labels Group → Labels внутри**.

## Кто использует

Назначают метки на треды — SDR и выше; создают группы и метки — уровни **1–3**.

## Как работает

Раздел — список **групп меток**; внутри каждой группы — свои **метки**. Сначала создаётся группа, затем в неё добавляются метки. Созданные метки назначаются на треды в Inbox.

**Создание группы — поп-ап.** Кнопка **+ New group** открывает небольшой модал: Group name (+ из TODO: режим Single/Multi, опц. описание). Редактирование группы (✎) — тот же поп-ап.

**Метки — через action «Add label» на группе (поп-ап).** У каждой группы — действие **Add label**; открывает модал (редактор метки уже не помещается inline). Раскрытая группа показывает свои метки чипами. Поля (цвет и AI — из TODO):

| Поле | Описание |
|---|---|
| Name | Название метки |
| Color | Цвет чипа (виден на треде в Inbox) |
| Auto-label with AI | Тогл: AI сам ставит эту метку на подходящие треды |
| AI instructions | Появляется, если тогл включён — **когда ставить эту метку** (как AI enrichment instructions в [Columns](/v4/workspace/settings/columns)); AI по этому описанию размечает треды |

| Labels Group | Labels |
|---|---|
| Stage | New · In progress · Won · Lost |
| Topic | Pricing · Demo · Support · Partnership |
| Priority | Hot · Warm · Cold |
