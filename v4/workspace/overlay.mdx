## Планируемые подстраницы

- Обзор — структура оверлея, Details panel
- Header-действия — Add task, Book meeting, Log call, Email, LinkedIn, ⋯ More (состав из THINK.md)
- Emails — треды, Reply / Reply to all с автозаполнением TO
- Activity — группировка по датам, фильтр по пользователю, timeline view
- Редактирование данных — inline-правка полей; добавление контакта: показывать ли автозаполненные поля для проверки перед сохранением (открытый вопрос)
