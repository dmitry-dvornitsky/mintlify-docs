Файлы и ссылки без папок: транскрипты звонков, sales-деки, PDF, таблицы, URL (сайт, кейсы, конкуренты). AI разбирает форматы сам и использует всё при генерации [документов](/v4/workspace/client/documents). Экран — список материалов; загрузка живёт в модале под кнопкой **Upload** ([D-013](/v4/decisions)).

## Кто использует

AS и AD загружают; вся команда видит. Клиент может присылать материалы через [Portal](/v4/portal) — они попадают сюда с пометкой «от клиента».

## Как работает

Materials — **вторая таба** модуля Client (первая — [Documents](/v4/workspace/client/documents): чтение чаще загрузки). Пока документы не сгенерированы, модуль открывается на Materials. Первая загрузка происходит на шаге 5 [визарда](/v4/admin/projects/create); дальше — в любой момент.

После добавления новых материалов система предлагает: «✦ Обновить документы с учётом новых материалов» — генерация идёт в фоне, по готовности приходит уведомление и открывается [последовательный ревью диффов](/v4/workspace/client/documents).

### Сайт клиента — системный материал

Сайт (домен из шага 1 [визарда](/v4/admin/projects/create)) — **первая строка списка**, создаётся системой и не архивируется: это источник №1 генерации, и он должен быть видим. В строке — дата последнего скана и действие **Rescan** (после редизайна сайта). В превью — что именно спарсилось. Uploaded by: System.

Список отсортирован по дате загрузки (новые сверху, системная строка сайта всегда первая), сортировка по любой колонке, ленивая подгрузка по 50 строк.

### Загрузка — модал под кнопкой Upload

Drop-зона и поле ссылки не занимают экран постоянно ([D-013](/v4/decisions)): список — главный контент страницы. Модал открывается кнопкой **Upload** или перетаскиванием файла в любое место экрана. Загрузка стартует сразу при добавлении; **Done** закрывает модал, статусы разбора дальше живут в таблице.

| Элемент модала | Действие |
|---|---|
| Drop-зона / **Browse** | Мультивыбор; валидация формата и размера до старта загрузки |
| Поле ссылки + **Add** | Добавляет URL в очередь разбора |
| **✕** у файла в очереди | Отмена незавершённой загрузки |
| **Done** | Закрывает модал; загрузки продолжаются в фоне |
| Дубликат (тот же hash) | Строка помечается «уже есть», копия не создаётся |

### Просмотр материала — оверлей, не страница

Клик по строке открывает **правую панель-оверлей** поверх списка (тот же паттерн, что [Company Overlay](/v4/workspace/overlay)): просмотр материала — быстрая проверка «что распарсилось», контекст списка терять нельзя. Отдельной страницы и URL у материала нет — ссылку на материал не шарят, шарят документы.

Внутри оверлея:

| Элемент | Действие |
|---|---|
| **↑ / ↓** в шапке | Соседний материал без закрытия оверлея (и с клавиатуры) |
| **✕** / Esc / клик мимо | Закрыть, вернуться к списку |
| **Open in document** (конфликт) | Переход в секцию документа со вторым значением |
| Подсветка в Parsed content | Место конфликта / место, использованное в документе |
| **Download original** | Скачать исходный файл |
| **Archive** | Как в таблице: подтверждение → архив (D-007) |
| Failed-материал | Вместо контента — причина разбора + **Retry** и подсказка, чем заменить |

## Правила и эдж-кейсы

| Ситуация | Поведение |
|---|---|
| Файл не разобрался (скан, битый PDF) | Статус Failed с причиной; материал остаётся, но помечен «не участвует в генерации» |
| Дубликат файла (тот же hash) | Предупреждение, повторная загрузка не создаёт копию |
| Архивирование материала ([D-007](/v4/decisions)) | Материал скрывается и исключается из **будущих** генераций; уже одобренные версии документов не меняются |
| Материал содержит противоречие с документами | AI при разборе помечает конфликт («в деке цена $99, в документе Продукт — $149») и показывает алерт AS |
| Ссылка недоступна (404, пароль) | Статус Failed, периодическая перепроверка, алерт при постоянной недоступности |
| Очень большой файл | Лимит размера и форматов с внятной ошибкой до начала загрузки |
| Материал от клиента через Portal | Бейдж «от клиента» + уведомление AS. Участвует в генерации сразу, без премодерации: документы всё равно меняются только через аппрув диффа |
| Транскрипты встреч из [Meetings](/v4/workspace/meetings) | **Не попадают в Materials автоматически** — это личная информация клиента. AS вручную загружает выбранные транскрипты |
| Очень длинный материал (сотни страниц) | AI автоматически строит выжимку для генерации, бейдж «summarized»; в превью — и выжимка, и полный текст. Полный текст остаётся в поиске |
| Архив | Фильтр **Archived** показывает архивные; разархивирование возвращает материал в будущие генерации. Сайт-строку архивировать нельзя |
