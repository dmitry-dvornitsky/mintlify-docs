## Планируемые подстраницы

- Companies — таблица, фильтры, bulk-операции
- Contacts — таблица, LinkedIn Score с датой анализа (THINK.md)
- Find — поиск компаний и контактов (аудитории в контексте поиска, не в Settings)
- Import — визард импорта (шаги раздельно: данные → настройки → валидация)
- История импортов — плоский список вместо Requests
- Списки (Lists) — создание, наполнение, использование в кампаниях
- DNC — уровни (контакт / компания / домен), приоритет над всем
- Дубликаты — обнаружение, merge, повторный импорт

## Вопросы к проработке

- Merge двух компаний: чьи данные побеждают, что с тредами и встречами обоих
- Контакт сменил компанию — история переезжает или остаётся
- GDPR-запрос на контакта — решено ([D-007](/v4/decisions)): персональные данные стираются через Admin → System, анонимная статистика остаётся
