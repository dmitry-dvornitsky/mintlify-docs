В V1 стратегию собирает Account Strategist через 5-шаговый wizard. В V3 наоборот: **AI собирает — человек выбирает**. Это работает одинаково в онбординге (первая стратегия) и после него (сколько угодно новых).

## Первый выбор (онбординг)

После аппрува документов Home ведёт на экран стратегий. AI уже сгенерировал **три карточки** на основе документов «Кому продаём» и «Что говорим»:

```
┌───────────────────┐ ┌───────────────────┐ ┌───────────────────┐
│ ⭐ Рекомендуем     │ │                   │ │                   │
│ CFO в строитель-  │ │ Операционные      │ │ Компании после    │
│ ных компаниях UK  │ │ директора в       │ │ раунда инвестиций │
│                   │ │ логистике EU      │ │                   │
│ Кому: CFO, Head   │ │ Кому: COO, VP Ops │ │ Кому: CEO, CFO    │
│ of Finance        │ │                   │ │                   │
│ Почему сработает: │ │ Почему сработает: │ │ Почему сработает: │
│ у них сезонный    │ │ ручной учёт при   │ │ есть бюджет и     │
│ кассовый разрыв — │ │ росте парка —     │ │ мандат на рост —  │
│ продукт закрывает │ │ боль из кейса №2  │ │ письмо в момент   │
│ его напрямую      │ │                   │ │ события           │
│                   │ │                   │ │                   │
│ ~1 200 компаний   │ │ ~800 компаний     │ │ ~50 компаний/мес  │
│ [Use strategy]    │ │ [Use strategy]    │ │ [Use strategy]    │
└───────────────────┘ └───────────────────┘ └───────────────────┘
        Не то? [✦ Suggest different strategies + поле пожеланий]
```

## Что в карточке

Только то, что нужно для выбора — на человеческом языке:

| Блок | Содержание |
|---|---|
| **Название** | Сегмент одной строкой |
| **Кому пишем** | Должности словами, не «Audience ID» |
| **Почему сработает** | 1–2 предложения с отсылкой к материалам клиента |
| **Размер рынка** | Примерное число компаний под критерии |
| ⭐ **Рекомендуем** | AI помечает одну карточку и объясняет почему |

## Что под капотом

Кнопка **Use strategy** создаёт полноценную Strategy-сущность из V1-редизайна — Goal, Company ICP, Contact ICP (с автоматически созданной Audience), Messaging, Personalization hook — целиком заполненную AI из документов клиента.

Ссылка **Show details** на карточке раскрывает эти 5 блоков в readonly-виде с кнопкой Edit — для тех, кто понимает, что делает. Новичку туда ходить не нужно.

| Правило | Как в V3 |
|---|---|
| Кампания без стратегии | Невозможна, как и в V1-редизайне |
| Audience | Создаётся автоматически внутри стратегии; отдельного раздела Audiences в V3 нет |

<Note>Три карточки, а не одна, — осознанно: выбор из готовых вариантов даёт чувство контроля и заставляет прочитать и сравнить, но не требует экспертизы. Одна кнопка «AI решил за тебя» снижает доверие, пустая форма — парализует.</Note>

## После онбординга — раздел Strategies

Постоянный раздел навигации. Все стратегии проекта карточками, у каждой — живые цифры:

```
┌─────────────────────────────────────────────────────┐
│ CFO в строительных компаниях UK          ● Active   │
│ Leads 480 · Campaigns 2 · Replies 6.1% · Meetings 7 │
│ [Find leads] [New campaign] [Change] [⋯]            │
├─────────────────────────────────────────────────────┤
│ Операционные директора в логистике EU    ⏸ Paused   │
│ Leads 120 · Campaigns 1 · Replies 1.8% · Meetings 0 │
│ [Resume] [Change] [⋯]                               │
└─────────────────────────────────────────────────────┘
   [✦ Suggest new strategies]     [Describe your own]
```

### Новые стратегии — AI учится на результатах

Кнопка **✦ Suggest new strategies** доступна всегда и с каждым разом становится умнее: AI смотрит не только на документы клиента, но и на накопленные результаты — какие сегменты отвечают, какие письма получают Positive, откуда пришли встречи. Карточка новой стратегии прямо ссылается на это: «Похожа на CFO UK (7 встреч), но сегмент шире — Ирландия и Скандинавия».

**Describe your own** — текстовое поле для своей идеи словами: «клиент просит попробовать финтех в Германии». AI собирает из этого полноценную стратегию-карточку — критерии, аудиторию, messaging — и показывает на аппрув как обычно.

### Изменение стратегии

Кнопка **Change** открывает стратегию как три простых блока — каждый можно перегенерить отдельно, словами:

| Блок | Пример правки |
|---|---|
| **Каким компаниям пишем** | «Добавь Ирландию, убери компании меньше 50 человек» |
| **Каким людям пишем** | «Добавь Financial Controller» |
| **Что говорим** | «Смени акцент с экономии на скорость закрытия месяца» |

AI применяет правку, показывает diff «было → стало», человек подтверждает. Полная V1-форма — по-прежнему под «Show details» для тех, кто хочет руками.

<Warning>Правка стратегии **не трогает активные кампании** — они работают на снапшоте (правило V1-редизайна). После изменения система предлагает: «Применить к кампании CFO UK / Jul?» — обновление письма коснётся только ещё не отправленных касаний. Новые поиски лидов и новые кампании используют новую версию автоматически.</Warning>

### Прочие действия (меню ⋯)

| Действие | Что делает |
|---|---|
| **Pause** | Стратегия не предлагается для поисков и кампаний; активные кампании продолжают идти |
| **Duplicate** | Копия для соседнего сегмента — AI сразу предлагает, что изменить |
| **Archive** | Скрыть выдохшуюся; история лидов и кампаний сохраняется |

<Note>Home продолжает помогать и здесь: если стратегия две недели даёт Reply rate ниже порога, в очереди появится карточка «Стратегия слабеет. Предлагаю: изменить messaging / предложить новые сегменты» — с кнопками, ведущими сюда. Но решение всегда за человеком.</Note>
